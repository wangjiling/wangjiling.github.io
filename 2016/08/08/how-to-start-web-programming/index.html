如何快速入门web前端开发

2016, Aug 08    

引语

大学读的专业是计算机科学与技术，计算机相关的课程（硬件、软件）基本都学了一遍。毕业后主要做web前端相关的开发工作，也使用过（java、php）等后端语言参与过后端开发，现在是专注web前端领域相关的技术开发和新技术的调研。

当下web相关技术和行业发展迅速，web前端开发工程师十分紧俏。在web前端开发道路上，喜悦、烦恼都历历在目，感慨颇多。后面会分享一系列web前端开发相关技术实践经验和技术干货，帮助系统性学习web前端开发知识。此篇与大家分享一下如何能更好地入门和系统性学习web前端相关技术知识，希望对初学Web前端的各位童鞋来说有所帮助。

切忌忽略基础

前面有提到过随着web迅速发展导致相关技术开发人员紧缺，但是，“缺”不代表web前端开发者少，而是满足招聘条件的优秀web程序员少。在不少程序员眼里web前端开发是一件很easy的事情，也就是写写标签、布下局外加一些简单的逻辑处理。不可否认，web入门相比较后端语言入门是要容易的多，但是想要全面深入地掌握是非常难的，丝毫不亚于对后端语言技术栈的掌握。 工作中遇见不少同行，有不少是半路出家转做web前端开发。技术好坏撇开不说，有一点可以确信，对基础重视和善于学习的人技术肯定不会太差。web前端相关技术框架层出不穷，bootstrap、backbone、angular、react、vue等等，很多初学者直接跨过基础知识的学习，直接使用框架进行开发，虽然也能很快实现看起来不错的应用（有些框架小白也能使用），但是没有扎实的基础写出来的代码的质量和性能可想而知。个人建议这类开发者还是要花一些时间回过头系统地学习下web基础知识，毕竟框架只是辅助工具，是为了更快速开发出高质量和高性能的应用。

开发工具使用

目前web前端开发比较热门的IDE有很多，例如Sublime Text、WebStorm、Brackets 、Eclipse等，通过配置可以极大地方便地我们的开发。 早期网页制作是用Dreamweaver，像这种所见即所得的IDE为我们制作网页带来了极大的方便。入门快、见效快让很多人陷入一个误区，既然借助这类IDE，通过鼠标点击菜单就可以快速方便地制作网页，那么我们为什么还要去学习HTML、CSS、JavaScrpt等这些繁琐的知识呢？但是随着学习的深入，就会发现我们步入了一种窘境——过分的依赖IDE导致我们不清楚其实现的本质，知其然但不知其所以然。因此在页面效果出现问题或者维护别人写的代码时变的手足无措，更不用提如何进行页面优化以及完成一些更高级的应用了。其原因是显而易见的——聪明的IDE成全了我们的惰性，使我们忽略了华丽的网页背后最本质的内容——code。 web初学者选择自己喜欢的IDE之后，前期避免使用里面的代码自动生成和自动补全功能，尽可能自己去写代码，写的过程就是一个思考的过程，很多初学者都是在coding的过程中快速打通任督二脉的。到了后期自己能够熟练掌握和书写web页面之后，可以慢慢放开使用IDE所具有的一些高级功能，让你的开发速度更上一层楼。

学习路线

这里只是针对web 前端基础技术知识的学习，不考虑框架和各种开发工具库，学习过程中要把相关基础概念理解透彻，这些都是后续建造高楼大厦的基石。

web前端基础技术大体包括三大部分：

  • html：页面结构内容层；
  • css：页面表现层；
  • js：页面行为层

（1＋2）html＋css：（推荐在线学习网站：w3school（http://www.w3school.com.cn/）或菜鸟教程（http://www.runoob.com/）；书籍：HTML5与CSS3权威指南） html＋css看起来比较简单，实则不然。初期必须掌握HTML的基本结构和常用标签及属性、HTML5新增标签和属性；了解HTML 4.01、XHTML 1.0、HTML5之间的差异；css学习重点：盒子模型，浮动，position，block、inline，样式优先级；css3新增样式属性。最后再深入了解下浏览器差异性，根据当前浏览器所占份额，ie8以下浏览器被淘汰掉是趋势，没必要浪费大量时间去学习ie低版本的兼容性，主要兼容的主流浏览器还有chrome浏览器、firefox浏览器、safari浏览器、opera浏览器即可。浏览器差异性内容很多，建议在实践中多多积累。

（3）javascript：（推荐在线学习网站：w3school（http://www.w3school.com.cn/）或菜鸟教程（http://www.runoob.com/）；书籍：《JavaScript高级程序设计》） javascript，难点，也是重点，要学的内容实在太多，如果没有其他编程语言的基础的话，学起来可能要费些力。首先要熟练掌握其基本语法和一些基本概念，另外必须要深入理解一些比较抽象的概念，比如闭包、原型、面向对象、作用域、封装等。了解es6新增概念和功能。掌握轻量级的文本数据交换格式JSON。看书一般是必不可少的，找一本优秀的js书从头到尾，反复研究学习，第一遍可以快速翻阅，大体知道了解概念即可，第二三遍就需要深入学习。（jQuery学习： js基础知识掌握好之后，还需要学jQuery，jQuery 是一个 JavaScript 库，封装了对DOM操作的一系列方法，消除了js跨平台兼容问题，并提供了大量实用方法，极大地简化了 JavaScript 编程，有良好的文档和帮助手册，很容易学习和使用。）

总结

随着互联网的快速发展，web相关技术也在不断迭代更新，对web前端开发工程师的要求也越来越高，不管是知识的广度还是深度。在学习的过程中要多看、多写、多查、多想，还要多浏览一些优秀的网站，善于分析借鉴其布局思路和实现方法，进而形成自己的一套技术知识体系。
